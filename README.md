# DNSFix
Este script .bat permite diagnosticar problemas comunes de red, tales como:  Falta de conexión a internet  DNS fallando  “Host no resuelto”  Mala configuración de red  Pruebas de ping  Detección de caídas de DNS públicos  Incluye herramientas automáticas para reparar DNS usando servidores públicos como Cloudflare (1.1.1.1) y Google (8.8.8.8).
